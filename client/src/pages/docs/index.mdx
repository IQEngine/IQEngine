# Getting Started

Add beautiful notifications to your React app with [react-hot-toast](https://github.com/timolins/react-hot-toast).

### Install with Yarn

```sh
yarn add react-hot-toast
```

### Install with NPM

```sh
npm install react-hot-toast
```
